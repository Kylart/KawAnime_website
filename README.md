# KawAnime's website

> [KawAnime](https://github.com/Kylart/KawAnime)'s official website.

## Build Setup

``` bash
git clone https://github.com/Kylart/KawAnime_website
 
cd KawAnime_website
 
npm install
 
# For dev mode
npm run dev
 
# OR 
 
# For production mode
npm run build && npm run start  # Or npm run bstart
```
